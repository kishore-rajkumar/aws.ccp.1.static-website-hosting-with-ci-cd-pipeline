Static Website Hosting with CI/CD Pipeline:

Demo: Website Link

Description: Build a simple static website (e.g., a personal portfolio, a landing page for a fictional business) and host it using Amazon S3 for storage and Amazon CloudFront for content delivery. Implement a CI/CD (Continuous Integration/Continuous Delivery) pipeline using AWS CodeCommit, AWS CodeBuild, and AWS CodeDeploy to automatically update the website whenever changes are pushed to the repository. AWS Services Used: S3, CloudFront, Route 53 (for custom domain, optional), AWS CodeCommit, AWS CodeBuild, AWS CodeDeploy, IAM.

Why it's valuable: Demonstrates understanding of scalable and cost-effective web hosting, version control, and automation in deploying updates.
